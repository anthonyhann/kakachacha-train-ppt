# AST Outline — 咔嚓咔嚓号小火车

## Audience
3-8岁儿童，注意力窗口约5-8秒/幕，
对运动、色彩、动物、友谊主题高度敏感。

## State Transfer Map

| 幕 | 情绪弧线 | 观众状态变化 | 核心动效钩子 |
|----|---------|------------|------------|
| S01 始发站 | 期待↑ | 好奇→兴奋 | 蒸汽喷出+小旗飘动 |
| S02 火车开动 | 活力↑↑ | 兴奋→投入 | 车轮旋转+背景流动 |
| S03 森林站 | 奇妙↑ | 投入→惊喜 | 小鸟扇翅+小花开合 |
| S04 告别森林 | 留恋→ | 惊喜→温柔 | 车窗视角+景物淡出 |
| S05 动物乐园 | 热闹↑↑↑ | 温柔→狂欢 | 动物群像+拍手节奏 |
| S06 夕阳归途 | 温馨↓ | 狂欢→安宁 | 光晕呼吸+火车剪影 |
| S07 学校站 | 圆满✦ | 安宁→感动 | 心形光斑放大淡出 |

## Narrative Arc（叙事弧）

```
情绪高度
  ↑
  |        S05★
  |      /     \
  |   S02        S06
  |  /    S03  S04   \
  | S01              S07✦
  +------------------------→ 时间
```

## Transfer Logic（每幕推进逻辑）

- **S01→S02**：鸣笛声触发"出发"，车轮开始转动，完成"静止→运动"的状态跃迁
- **S02→S03**：速度感渐缓，森林站牌出现，"运动→到达"的节奏切换
- **S03→S04**：车门关闭，视角切入车窗内，"外部世界→内心感受"的视角转换
- **S04→S05**：节奏加快，动物叫声提前入场，"留恋→期待新惊喜"
- **S05→S06**：光线变暖，动物渐远，"热闹→安静"的自然过渡
- **S06→S07**：火车减速，灯光亮起，"旅途→归宿"的情感收束
- **S07 结尾**：心形光斑是全片情感最高点，用"慢"对抗"快"，制造记忆锚点

## Core Design Principles

1. **蓝色火车一致性**：每幕必须出现同款蓝色老式蒸汽火车，强化品牌记忆
2. **群像人物连贯性**：相同的儿童角色贯穿全程，观众建立情感连接
3. **动效优先于文字**：字幕仅作辅助，动效是主要叙事手段
4. **每幕独立可循环**：每幕动效设计为可循环播放，适应演出节奏调整
